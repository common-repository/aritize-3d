== Installation ==

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended

= Manual installation =

1. Upload the entire `aritize-3d` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Aritize 3D** menu in your WordPress admin screen.

Go to the Aritize 3D -> Settings, click Generated token
